# Python-Conditional-Statements-Examples
This repository contains **5 beginner-friendly Python examples** demonstrating how to use ** some conditional statements** (`if`, `elif'', 'else`) in different scenarios, including grades, person, age, and school classes.
## What You Will Learn
By studying this Python file, you will learn:
- How to use **if, elif, and else statements** to check conditions
  This file contains:
1. A simple `if` statement (Grade check)  
2. An `if-else` statement (Pass/Fail)  
3. An `if-elif-else` statement (Grade classification)  
4. Age classification (Child, Teen, Adult)  
5. School class check (JSS1, JSS2, JSS3)
## Notes for Readers
There is a word document which explains all the types of condition statements.
## How to Run
1. Make sure Python is installed on your computer.  
2. Clone this repository.  
3. Open `conditionals.py` in any Python IDE or text editor.  
4. Run the file to see the output of each conditional example.
